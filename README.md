## Traffic Light Control using 8051 (Keil + Proteus)

Microcontroller: AT89C51  
IDE: Keil µVision  
Simulation: Proteus  

Description:
This project simulates a traffic signal using Port 1 pins.

Connections:
P1.0 → Red LED  
P1.1 → Yellow LED  
P1.2 → Green LED  

The program sequentially activates LEDs using software delay.
